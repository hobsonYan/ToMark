# ToMark - 轻量网页批注工具

## 简介

ToMark 是一款简洁高效的网页批注工具，让你在任何网页上轻松高亮文本、添加笔记。所有数据保存在本地，隐私安全，即装即用。

## 功能特点

### 📝 划词高亮
选中文字，一键高亮。支持4种颜色：
- 黄色 - 重要标记
- 绿色 - 观点想法
- 蓝色 - 待办事项
- 粉色 - 个人备注

### 📌 添加批注
为高亮添加笔记说明，方便回顾和整理。

### 💾 本地存储
- 数据保存在浏览器本地
- 刷新页面批注不丢失
- 无需注册登录
- 完全离线可用

### 🎨 支持重叠高亮
同一段文字可以添加多层不同颜色的高亮，分类更清晰。

### 🔄 颜色修改
随时修改已有高亮的颜色，轻松调整分类。

### 📤 一键导出
将所有批注导出为文本文件，方便备份和整理。

### ⌨️ 快捷键
- `Ctrl+Shift+H`：快速高亮选中文本

## 使用场景

- 📚 阅读学习 - 标记重点、添加理解
- 📰 新闻阅读 - 标注要点、整理资讯
- 💼 工作文档 - 标记待办、记录灵感
- 📑 研究资料 - 整理文献、标注引用

## 安装

1. 下载并解压
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 ToMark 文件夹
