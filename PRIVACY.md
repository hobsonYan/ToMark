# 隐私政策

**最后更新：2026年**

## 简介

ToMark（"我们"、"我们的"）高度重视您的隐私。本隐私政策说明了我们如何处理您的数据。

## 数据收集

**我们收集什么：**
- 您高亮的文本内容
- 您添加的批注
- 您访问的网页 URL
- 页面标题

**我们如何存储：**
- 所有数据仅存储在您浏览器的本地存储（Chrome Storage）中
- 数据不会上传到任何服务器
- 数据不会与任何第三方共享

## 数据使用

我们使用收集的数据仅用于：
- 在您访问的网页上显示您的批注
- 在弹窗中展示您的批注列表
- 导出功能

## 数据共享

**我们不分享、不出售、不转让您的任何数据给任何第三方。**

## 数据控制

您拥有数据的完全控制权：
- 可以随时删除单条批注
- 可以一键清空所有数据
- 可以导出数据备份

## 权限说明

ToMark 请求以下权限：
- `storage`：保存您的批注数据
- `activeTab`：在当前页面显示批注
- `<all_urls>`：访问所有网页以显示批注

## 政策更新

我们可能不时更新本政策。更改将在 GitHub 仓库中发布。
